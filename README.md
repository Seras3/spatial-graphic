# 🪐spatial-graphic 
**Solar System** representation in OpenGL.


Dependency: **glew32.lib**; **freeglut.lib**

# Controls ⌨️

**R-Click**   👉 Anticlockwsie Planet Rotation

**L-Click**   👉 Clockwise Planet Rotation

**SPACE**     👉 Launch Rocket

**↑ ↓ ← →**   👉 Explore The Space With 🚀
